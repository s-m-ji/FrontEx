6. color

텍스트의 색상을 지정할 때 사용
  • 색상명으로 지정 ▶ cornflowerblue
  • 16진수 값으로 지정 ▶ #6495ed
  • reg로 설정 ▶ rgb(100, 149, 237)
  • rgba로 설정 ▶ rgba(100, 149, 237, 1)
  • hsl로 설정 ▶ hsl(219, 79%, 66%)
  • hsla로 설정 ▶ hsla(219, 79%, 66%, 100%)

8. text-shadow

텍스트에 그림자 효과를 추가
text-shadow : 가로거리, 세로거리, 번짐정도, 색상
양수 : 글자의 오른쪽, 아래쪽
음수 : 글자의 왼쪽, 위쪽
번짐 정도 : 양수값 그림자가 모든 방향으로 퍼져 나가므로 크게 표현
눕고 싶어요
