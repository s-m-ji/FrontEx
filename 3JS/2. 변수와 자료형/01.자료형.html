1) 자료형

      자바스크립트에서는 변수에 별도로 자료형을 지정해주지 않고 [리터럴]에 의해
      자료형이 결정된다.
      * [리터럴] : 변수에 대입되는 값 그 자체를 의미
  
        - 자료형을 확인하는 방법
        typeof (값 또는 변수명)

Primitive type (원시 타입)

하나의 값만 저장하는 자료형

  • srting(문자열)

    ''. ""로 묶은 데이터

  • number(숫자)

    정수와 실수 모두 포함

    * 숫자가 들어있어도 따옴표로 묶이면 문자열로 인식

  • boolean (논리값)

    true / false

                - truthy : true로 인정할 수 있는 값 (falsy를 제외한 값)
            - falsy : false로 인정할 수 있는 값
                => 0, "", '', `` 빈 문자열, NaN, undefined, null
                * Boolean(0); // Boolean type으로 형변환
                예시) res = 0?'true':'false' (*0? : falsy값이라면?을 의미)
  • undefined

    변수를 선언만 하고 값을 할당하지 않았을 때 변수의 초기값

  • null

    값이 없거나 유효하지 않은 값

2. 데이터 형변환

1) 문자열과 숫자의 '+' 연산

        문자열 + 숫자열 = 문자열
        숫자 + 숫자 + 문자열 = 문자열 (숫자가 먼저 계산되고 뒤의 문자열과 합)

        문자형과 + 연산 시 문자열로 변경
        그 외의 연산 (-, * 등등) 시 숫자로 변환하여 연산 처리함

2) 강제 형변환

        Number(값), parseInt(값), parseFloat(값)
        문자형, 논리형 -> 숫자형으로 변환

        String(값), 값.toString
        숫자형, 논리형 -> 문자형으로 변환

        Boolean(값)
        문자형, 숫자형 -> 논리형으로 변환

3. 연산자

'==' 연산자와 '===' 연산자

        '=='은 값이 일치하는지 확인하는 연산자.
        '==='은 값과 자료형이 일치하는지를 확인하는 연산다.
