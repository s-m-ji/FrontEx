함수

1. 함수 선언

1) 선언적 함수

function 키워드에 함수명을 작성하여 선언하는 방법

2) 익명 함수

함수명을 작성하지 않고 변수에 대입하거나 인수로 넘겨주는 방법이다.

3) 즉시 실행 함수 (스스로 실행하는 함수)

함수 정의와 동시에 즉시 실행되는 함수로 한 번만 호출되며 다시 호출할 수 없다.

4) 화살표 함수(Arrow Function)

        ES6에 도입된 함수 선언 방법
        function 키워드 대신 => 사용해 좀 더 간략한 방법으로 함수를 선언적

        [표현법]
                                ※ function() => { ... }
            - 매개 변수가 없을 때 : () => { ... }    
            - 매개 변수가 하나일 때 : a => { ... }    
            - 매개 변수가 두개 이상일 때 : (a, b) => { ... }    

            - 리턴 처리
            - 처리할 라인이 여러 개라면 { }로 감싸서 처리해야한다.
            (a, b) => {
                실행 구문
                return a * b;
            }
            - 처리할 라인이 한 줄이라면 { }와 return을 생략할 수 있다.
            (a, b) => a * b;

2. 함수의 매개변수

1) 매개 함수

    함수 실행에 필요한 값을 외부에서 함수로 전달할 때
    매개변수를 통해서 매개값을 전달한다.
    함수를 호출할 때 매개 변수를 개수만큼 전달하는 것이 일반적이지만
    그렇지 않은 경우에도 에러가 발생하지 않는다.

2) arguments

    초과된 매개값이나 가변적으로 매개값을 함수 내부로 전달할 때는 arguments 객체를 사용한다.
    (모든 매개값이 arguments 객체의 프로퍼티로 보관된다.)

3. 함수의 리턴

1) 일반적인 값 리턴

2) 익명 함수를 리턴하는 함수

4. 내장 함수

1) eval() 함수

문자열을 자바스크립트 코드로 변환해서 실행하는 함수

2) isInfinity()와 isNaN() 함수

        자바스크립트의 경우 어떤 수를 0으로 나누면 infinity라는 값이 들어가게 되고
        자바스크립트가 표현할 수 없는 숫자는 NaN(Not a Number)로 표시된다.
