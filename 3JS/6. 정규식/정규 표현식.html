정규 표현식

         정규 표현식
         특정 규칙을 가진 문자열을 검색하거나 치환할 때 사용하는 형식
         주로 전화번호, 이메일, 파일 확장자의 형식을 확인할 때 사용함

1. 정규 표현식 객체 생성

      정규 표현식 리터럴
      RegExp 생성자 함수

2. 플래그 문자

      정규 표현식의 검색 방식을 지정
      순서와 상관없이 여러 개를 지정할 수 있다
  • i : 대소문자를 가리지않고 비교
  • g : 전역으로 비교를 수행
  • m : 여러 줄의 비교를 수행

3. 메타 문자

      특정 메타 문자를 통해 정규식으로
      특정 패턴을 제시할 수 있다
