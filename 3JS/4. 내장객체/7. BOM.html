BOM(Browser Object Model)

window 객체

window 객체는 자바스크립의 최상위 객체이며, 브라우저 내의 모든 요소들과 객체를 포함한다.

1) window.open()

2) 타이머 관련 메소드

특정 함수를 일정 시간이 경과된 이후에 호출할 수 있도록 제공된다.

window.setTimeout()

일정 시간이 경과된 이후에 매개값으로 전달된 콜백 함수를 단 한 번만 실행한다.

window.setInterval()

일정 시간이 경과된 이후에 매개값으로 전달된 콜백 함수를 반복해서 실행한다.

BOM(Browser Object Model) 객체

1) location 객체

브라우저의 주소 표시줄과 관련된 객체

location reload()

페이지를 새로고침 하는 메소드

location.href

        현재 페이지의 URL 주소를 가지고 있는 속성으로 
        해당 속성의 값을 변경해서 페이지를 이동할 수 있다.

location.assign()

location.replace()

history에 기록되지않음

2) history 객체

브라우저의 방문 기록과 관련된 객체

3) navigator 객체

웹 페이지를 실행하고 있는 브라우저에 대한 정보를 가지고 있는 객체

4) screen 객체

운영체제의 화면(모니터) 정보를 가지고 있는 객체
